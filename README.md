# Multi-Channel Marketing Mix Analysis using Multiple Linear Regression

## Project Overview
This project builds a statistically sound Multiple Linear Regression model using Python and `statsmodels` to analyze and evaluate performance across three distinct marketing channels: **TV, Radio, and Social Media**. The primary objective is to optimize budget allocations based on verified feature significance and regression coefficients.

## Key Goals
1. Check and correct for multicollinearity using Correlation Matrices and Variance Inflation Factors (VIF).
2. Execute an Ordinary Least Squares (OLS) multivariate regression.
3. Validate linear regression assumptions (Linearity, Normality, and Homoscedasticity) via diagnostic plots.
4. Formulate an evidence-based recommendation for marketing budget allocation.
